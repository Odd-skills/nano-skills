# 提示词模板

预设的提示词模板，覆盖常见图片生成场景。

## 原型图设计

| 场景 | 模板 |
|------|------|
| 移动端界面 | `设计一个 [功能描述] 的移动端App界面原型，简洁现代风格，深色/浅色主题` |
| Web 页面 | `设计一个 [类型] 网站的首页原型，响应式布局，包含导航栏、主图、功能区` |
| Dashboard | `设计一个数据分析仪表盘界面，包含图表、统计卡片、侧边栏导航` |
| 登录页面 | `设计一个简洁的登录页面，包含邮箱/密码输入框和社交登录按钮` |

### 示例

```
设计一个外卖订餐App的主页界面原型，包含：
- 顶部搜索栏
- 分类导航图标
- 推荐商家列表
- 底部导航栏

风格：简洁现代，浅色主题，圆角卡片设计
```

## 多图融合

| 场景 | 模板 |
|------|------|
| 风格迁移 | `使用第一张图片的艺术风格，重新绘制第二张图片的内容` |
| 元素合成 | `将第一张图片中的 [元素] 合成到第二张图片的 [位置]` |
| 场景融合 | `将两张图片的场景自然融合，创造一个连续的画面` |

### 示例

```
使用第一张图片（梵高星空）的绘画风格，
重新绘制第二张图片（城市街景）的内容，
保持梵高标志性的漩涡笔触和色彩运用
```
